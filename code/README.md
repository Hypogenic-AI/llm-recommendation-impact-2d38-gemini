# Cloned Repositories

1. [MusiCRS](musiCRS/)
   - URL: https://github.com/rohan2810/musiCRS
   - Purpose: Benchmark for audio-centric conversational recommendation.
   - Key features: Audio-LLM baselines, evaluation scripts, dataset processing.

2. [LLM4RS](LLM4RS/)
   - URL: https://github.com/rainym00d/LLM4RS
   - Purpose: Implementation of "Uncovering ChatGPT’s Capabilities in Recommender Systems".
   - Key features: Point-wise, pair-wise, and list-wise ranking prompts for LLMs.

3. [Microsoft Recommenders](recommenders_team/)
   - URL: https://github.com/recommenders-team/recommenders
   - Purpose: Comprehensive collection of recommendation algorithms.
   - Key features: Baselines like LightGCN, BPR, and various content-based/collaborative filtering methods.
